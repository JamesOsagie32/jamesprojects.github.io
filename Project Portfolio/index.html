Hospital Transactions Analysis

This project aims to visualize the opeating performance of DeepCare Hospital in order to determine if they are making profit or loss. From the analysis, we can see that the hospital's total revenue is higher than the total expenses which indicates it is operating at a profit level. We were also able to show which hospital specialty, hospital category and hospital procedures were generating the highest revenue. This analysis was done using PowerBi.
